Keyboard Shortcuts

OSSI/Sungard style Alt commands for quick CAD operations

Unit Status Commands

Alt+DDispatch unit
Alt+EEnroute
Alt+AArrived on scene
Alt+OOn scene (same as A)
Alt+TTransport
Alt+CClear unit
Alt+BBack in service

Incident & Navigation

Alt+NNew incident
Alt+IOpen incident
Alt+KAdd remark
Alt+HHeld calls
Alt+LDaily log
Alt+SSearch / History
Alt+UUnits panel
Alt+RSend report
Alt+FFocus command line

Function Keys

F1This help screen
F2New incident
F5Refresh panels
F9Daily log
EscClose modal / Cancel
HHeld calls (when not typing)
Tip: Alt shortcuts work even when typing in input fields. Press Alt+F to quickly focus the command line.
